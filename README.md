# Keypass AI
An open-source, API-first AI control plane that lets enterprises
evaluate, simulate, route, and pay for AI usage across multiple models
— without vendor lock-in.
# What this repo demonstrates**
- AI feasibility scoring before you build
- Multi-model routing per task
- Pay-per-use AI execution
- ROI and cost visibility
- Vendor-neutral architecture
# Extending this repo
- Add a new AI provider
- Add a scoring rule
- Add a policy constraint
⚠️ This is a reference implementation, not a full product.
